## About The Project
<br/><em><b>Clean Blog</b></em> with a number of blog posts that are retrieved through requests from the <a href="https://api.npoint.io"> npoint api</a> and render the different post content, <em></em><b>about page,contact form</b><em></em> in a Blog that is based on the <a href="https://startbootstrap.com/previews/clean-blog"> Clean Blog template.</a>  With the functionality to sends an email automatically when the contact form is submitted.
